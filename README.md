Crystal Beauty – E-Commerce Web Application

Crystal Beauty is a full-stack e-commerce web application developed using the MERN stack (MongoDB, Express.js, React.js, Node.js).

This platform allows users to browse products, register accounts, and place orders easily, while providing administrators with full control over product and user management.

🚀 Features

👤 User Features
User registration and login system
Browse and view product details
Product overview page
Place orders 
Secure authentication with password hashing


🔑 Admin Features
Add new products
Update existing products
Delete products
View all products
View all user accounts
View and manage all orders

🔒 Security
Implemented password hashing for secure authentication
Protected user data and login system

🖼️ Image Handling
Product images are stored using Supabase storage platform


🛠️ Tech Stack
Frontend: React.js
Backend: Node.js, Express.js
Database: MongoDB
Storage: Supabase
